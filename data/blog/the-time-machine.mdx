# Certified Bug Bounty Hunter

HTB Certified Bug Bounty Hunter (HTB CBBH) es una certificación altamente práctica
que evalúa las habilidades de búsqueda de recompensas de errores y pentesting de
aplicaciones web de los candidatos. Los titulares de la certificación HTB Certified
Bug Bounty Hunter poseerán competencia técnica en los dominios de búsqueda de
recompensas de errores y pruebas de penetración de aplicaciones web en un nivel intermedio.
Podrán detectar problemas de seguridad e identificar vías de explotación que pueden
no ser inmediatamente evidentes al buscar CVE o PoC de exploits conocidos.
También pueden pensar de manera innovadora, encadenar múltiples vulnerabilidades para
mostrar el máximo impacto y ayudar de manera práctica a los desarrolladores a remediar
las vulnerabilidades a través de informes de errores de nivel comercial.

### Público objetivo

- Cazarrecompensas de errores de nivel básico
- Probadores de penetración de aplicaciones web junior
- Desarrolladores web

### Requisitos previos

La siguiente es una lista de requisitos previos para un resultado exitoso:

- Interpretar una carta de compromiso y tener conocimientos intermedios sobre aplicaciones web, servicios web y pruebas de penetración de API.
- Conocimiento sobre aplicaciones web, servicios web y fundamentos de API.
- Realización de análisis estáticos y dinámicos de aplicaciones web/servicios web.
- Realización de identificación y análisis de vulnerabilidades de aplicaciones web, servicios web y API.
- Realización de explotación manual y automatizada de varias clases de vulnerabilidad.
- Comunicar y reportar vulnerabilidades profesionalmente

Extraido de la pagina de [HackTheBox Academy](https://academy.hackthebox.com/exams/2/).
